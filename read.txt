1. Tìm hiểu về dữ liệu
1.1 Folder train 
- train_sequences.fasta: cung cấp các chuối protein với 2 thành phần chính: EntryID, sequence
- train_terms.tsv: Nhãn GO cho protein train(EntryID,term,aspect)
    + term là nhãn GO cụ thể, còn aspect nói nhãn đó thuộc MF/BP/CC.
    + term = mã GO term (nhãn chức năng, lablel cần dự đoán)
    + aspect : GO term đó thuộc nhanh Ontology nào trong Gene Ontology
        * MF: Molecular Function (chức năng phân tử: binding,enzyme activity )
        * BP: Biological Process (quá trình sinh học: signaling, metabolism…)
        * CC: Cellular Component (vị trí trong tế bào: nucleus, membrane…)
    + train_taxonomy.tsv: thông tin taxonomy/species của protein train
        * Taxonomy: hệ thống phân loại sinh vật theo cấp bậc
        VD: TaxID=9606 - Homo spaiens 
        TaxID=562 - E.coli
        * species chính là loài
    + go-basic.obo: Gene Ontology(cấu trúc GO, quan hệ cha-con)
1.2 Folder test
- testsuperset.fasta: (EntryID + taxonID) ở header + sequence amino acid
- testsuperset-taxon-list.tsv: file TSV liệt kê danh sách taxon/species xuất hiện trong tập test (một “taxon list” cho test).\
sample_submission.tsv: template để nộp
IA.tsv: thông tin IA cho GO terms 
    * Information Accetion: thông tin/độ đặc hiệu tăng thêm của từng GO term
    * GO term càng chung chung -> IA càng nhỏ
    * GO term càng cụ thể -> IA lớn
1.2 file fasta  
- Là một định dạng file văn bản (text) dùng để lưu trình tự sinh học
- protein: chuôi amino acid (A,C,D,E,F ...)
- DNA/RNA: chuỗi nucleotide (A,T/U, C, G)
- Cấu trúc một bản ghi fasta gồm: Header + Sequence
2. THư viện ESM
- Là thư viện "ngôn ngữ protein"
- Do meta AI phát triển
- dùng Trainsformer để học từ chuỗi amino acid như BERT/GPT học từ văn bản
2.1 Công dụng
- Biến protein sequence thành embedding vector. sử dụng để phân loại/dự đoán
- Dùng cho các tác vụ như
    + Dự đoán chức năng
    + Tìm protein tương tự
    + Dự đoán đột biến ảnh hưởng như nào ...
2.2 Cách cài
pip install fair-esm

Gói này cung cấp code để load các model ESM pretrained và chạy inference lấy embedding cho chuỗi protein.
